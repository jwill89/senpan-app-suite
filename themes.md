# Theme CSS Overrides

Each theme below contains only `:root` variable overrides. Copy the CSS for a theme into the Themes editor in the admin panel.

---

## Main Light Theme

```css
:root {
    --bg: #d5bdaf;
    --surface: #e3d5ca;
    --surface2: #d1c4b5;
    --primary: #6b705c;
    --primary-hover: #5a5e4c;
    --secondary: #8a8568;
    --secondary-hover: #7a7558;
    --gold: #6b705c;
    --text: #2c2c2c;
    --text-dim: #5e5a50;
    --text-on-primary: #f0ebe3;
    --text-on-toast: #f0ebe3;
    --success: #6b705c;
    --danger: #a44a3f;
    --board-cell: #f5efe8;
    --board-cell-hover: #ece4da;
    --board-cell-free: #c4a999;
    --board-gradient-start: #cbbcae;
    --board-gradient-end: #bfae9e;
    --overlay: rgba(0, 0, 0, .5);
    --shadow-color: rgba(0, 0, 0, .2);
    --glow-color: rgba(107, 112, 92, .4);
}
```
